# 📚 词性突击 · 四级词缀速查与闯关

[![GitHub Pages](https://img.shields.io/badge/demo-online-brightgreen)](https://你的用户名.github.io/cet4-pos-master/)

> 一个专为英语四级考生设计的词性记忆与闯关工具，通过「后缀规律」和「实战线索」快速判断单词词性。

## ✨ 特色功能

- 🔑 **黄金口诀** — a/an/the后必填名词、主语宾语间必填动词、完整句子后99%副词
- 🧬 **后缀基因库** — 名词/动词/形容词/副词四大类后缀速查表，420+四级高频词
- 📘 **情态动词速查** — 点击悬浮窗一键查阅四级必考情态动词（含例句）
- 🎲 **词性闯关** — 系统词库 + 个人词库双模式，答对自动下一题，进度自动保存
- 🌐 **中英文切换** — 一键切换界面语言，中英文学习者皆可友好使用
- 📊 **结算分享** — 一键生成本次学习报告卡片，可下载PNG分享至社交平台
- 📱 **响应式设计** — 手机/平板/电脑均可流畅使用，随时随地备考

## 🚀 快速开始

## 📖 使用说明

| 模块 | 操作说明 |
|:---|:---|
| **学习模式** | 浏览口诀卡片、后缀表、线索表格，理解词性判断逻辑 |
| **测验模式** | 点击右下角🎲按钮，选择单词的正确词性 |
| **题库切换** | 在测验界面可切换「系统词库」或「我的词库」 |
| **添加单词** | 在我的词库模式下，点击「➕ 添加单词」自定义词条 |
| **情态动词** | 点击右下角📘按钮，弹出四级情态动词列表 |
| **重置进度** | 测验统计栏右侧点击 `↻` 清零当前题库答题记录 |
| **结算分享** | 点击「📊 结算分享」生成学习报告，可下载为图片 |

## 🧠 词库说明

系统词库包含 **420+** 个四级真题高频词汇，严格遵循后缀规律：

- **名词后缀**：-tion, -ment, -ness, -ity, -cy, -th, -ship, -hood, -er, -or, -ist...
- **动词后缀**：-ate, -ize, -ify, -en, -ish...
- **形容词后缀**：-ful, -less, -ous, -able, -ive, -al, -ic, -y, -ent...
- **副词后缀**：-ly, -ward(s), -wise

特别收录 `friendly / lovely / lively` 等 **-ly 结尾但为形容词**的易混淆词。
